Usage

Html

Add the socialshare attribute to any of your elements

Javascript

Add the socialshare module dependency
angular.module('app', [ '720kb.socialshare' ]);

Options

Sharing Provider

Set the platform you want to share on, use only lowercase (twitter, facebook, google+, vk, stumbleupon, digg, delicious, reddit, pinterest, tumblr)

Sharing Url

You can use the socialshare-url="" attribute to set the url to share

Sharing Content

You can use the socialshare-text="" attribute to set the content to share

Sharing Media

You can use the socialshare-media="" attribute to set the media to share (for example, you will use it for pinterest)

Sharing Hashtags

You can use the socialshare-hastags="" attribute to set the hashtags to share (for example, you will use it for twitter)

Sharing Via

You can use the socialshare-via="" attribute to set the twitter "via" option tag to share

Sharing Event Trigger

You can use the socialshare-trigger="" attribte to set the event trigger to share (you can use all the element.bind() accepted triggers of angularjs)

Sharing Popup size

You can use the socialshare-popup-height="" and socialshare-popup-width="" attributes to set sharer popup sizes
